webGL

WebGL (Web-based Graphics Library) — багатоплатформовий API для 3D-графіки в браузері, що розробляється некомерційною організацією Khronos Group. WebGL використовує мову програмування шейдеров SGSL. WebGL виконується як елемент HTML5 і тому є повноцінною частиною об'єктної моделі документа (DOM API) браузера. Може використовуватися з будь-якими мовами програмування, які вміють працювати з DOM API, наприклад, JavaScript, Rust, Java, Kotlin [3] та іншими. Всі провідні розробники браузерів Google (Chrome), Opera (Opera), Mozilla (Firefox), і Apple (Safari), є членами Khronos і реалізують WebGL в своїх браузерах. За рахунок використання низькорівневих засобів підтримки OpenGL, частина коду на WebGL може виконуватися безпосередньо на відеокартах. WebGL - це контекст елемента canvas HTML, який забезпечує API 3D графіки без використання плагінів. Перша специфікація була випущена 3 березня 2011 року.
