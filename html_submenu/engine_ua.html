cocos


Офіційний сайт розробника: www.cocos2d-x.org

Найбільш необхідна версія саме Cocos 2D-X, завдяки C ++ і крос-платформеності. Поточної стабільної версією рушія є 2.1.5, яка вийшла 23 серпня цього року. Заявлена підтримка: iOS, Android, Windows Phone 7 (XNA), BlackBerry, Tizen, Bada, Marmalade, Windows, Linux. Використовуючи Cocos 2D-X, можна кодувати на C ++, Lua і JavaScript. Рушієм користуються як програмісти-дослідники, невеликі інді-команди, так і монстри ігрової індустрії: Zynga, Konami, Disney Mobile. Завдяки шаленій популярності рушія - півтора мільярда закачувань. В його розвитку беруть участь розробники з Google, Microsoft, Intel.

coronaSDK


Офіційний сайт розробника: coronalabs.com

Corona SDK надає інструменти для створення крос-платформних ігор. API дозволяє створювати 2D гри, а також допоможе вам створювати складні функції, використовуючи API на основі Lua. Крім того, ви можете монетизувати Corona SDK з допомогою Corona Ads. Розробка за допомогою Corona досить проста завдяки тестуванню в режимі реального часу. приклад розробки за допомогою CORONA SDK.

CryEngine


Офіційний сайт розробника: mycryengine.com

CryEngine 3 орієнтований на створення крос-платформних ігор, призначених для PC і консолей. В даний час підтримуються платформи Xbox 360, Xbox One, PlayStation 3-4, WiiU, а також технології візуалізації настільної Windows - DirectX 9-11. Як можна помітити, підтримки мобільних платформ немає. У ньому присутня підтримка глобальних мультиплеєрних (MMO) ігор. CryEngine 3 володіє приголомшливим списком технологій візуалізації, ось деякі з них: динамічне освітлення і затінення в реальному часі, затуманення, Terrain 2.5D, карти нормалей і паралакс-маппінг, підповерхневе розсіювання, світлові промені і хвилі, управління рівнем деталізації ландшафту, а також багато іншого. Найцікавіше, що CryEngine за своїми можливостями випереджає поточну версію DirectX.

GameSalad


Офіційний сайт розробника: gamesalad.com

GameSalad ще один рушій для розробки гри з мінімальними або взагалі без навичок кодування. Він простий у використанні, і будь-яка людина може створити гру з нуля. Ще однією перевагою використання GameSalad є той факт, що вона дозволяє опублікувати гру на всіх відомих платформах, включаючи Windows, Android, IOS, і т.д. Плюс до всього, рушій доступний як для Mac так і і Windows.

heroengine


Офіційний сайт розробника: www.heroengine.com

Історія рушія розпочалася ще в кінці дев'яностих, коли американська компанія Simutronics приступила до розробки MMORPG Hero's Journey. Як показав час, із грою у них нічого не вийшло: багаторазові переноси дати і зриви виходу проекту. Однак технологія, на основі якої велася розробка, була високо оцінена всередині індустрії і на ігрових заходах, і рушій був ліцензований декількома студіями, у тому числі BioWare. У підсумку в 2010 році движок як самостійна технологія була продана корпорації Idea Fabrik. У цьому ж році вона почала продаж рушія під двома ліцензіями. Ліцензія передбачає покупку усього рушія разом з вихідними кодами за нечувану суму грошей. Користувачеві доведеться організовувати всю апаратну і програмну інфраструктуру гри. Крім серверного заліза, сюди входить: серверна ОС (Cent OS), БД Oracle, ліцензія на яку коштує дорого. Виходячи з другого варіанта, усі ліцензії на додаткове ПО, обслуговування серверного ПО і заліза бере на себе Idea Fabric - для інді-студій дуже прийнятний варіант.

libGDX


Офіційний сайт розробника: libgdx.badlogicgames.com

libGDX це кросплатформенних фреймворк який написаний на Java і працює з OpenGL. Пропонує підтримку декількох платформ для публікації гри. Ingress (попередник Pokémon GO) був розроблений з використанням libGDX. Підтримка спільноти також чудова, завдяки цьому ви можете отримати відмінну документацію для роботи. Приклад розробки з використанням libGDX.

Phasher


Офіційний сайт розробника: phaser.io

Phaser це HTML5 движок с відкритим вихідним кодом, який в останній час створює багато розголосу. Він добре підтримується Ричардом Дэви і його спільнотою. Це гарний варіант для тих, хто хоче розробляти веб ігри.

torque


Офіційний сайт розробника: www.garagegames.com

На ньому можна розробляти гру для трьох платформ: Windows, OS X і, що найцікавіше, iOS (і продавати ігри в App Store, не відраховуючи ні копійки авторам рушія). Весь рушій - це одна кодова база на C ++ без додаткових експортерів. Поточною стабільною версією є 2.0. Як видно, фундаментальні відмінності 2D- і 3D-версій полягають у графічній підсистемі: T2D для візуалізації використовує OpenGL, а T3D - DirectX, притому ще поки дев'ятої версії, що перегороджує створеним з його допомогою іграм шлях у Windows Store. Є куди розвиватися!

turbulenz


Офіційний сайт розробника: biz.turbulenz.com

HTML5 рушії користуються популярністю серед розробників ігор. Один з таких Turblenz, відкрита платформа для розробників ігор. Він включає в себе всі основні функції, які необхідні щоб розробити, інтегрувати і монетизувати гру. Крім того, немає ніяких обмежень у використанні, так як він доступний за ліцензії MIT.

udk


Офіційний сайт розробника: www.unrealengine.com/udk

UDK — це безкоштовна версія рушія UE 3, що володіє всім успадкованим інструментарієм останнього для створення ігрових світів. Список підтримуваних платформ не так широкий, як у Unity, але цього цілком вистачає, щоб окупити розробку: Windows PC, Windows Store, OS X, iOS, Android і консолі передостаннього покоління. Для скриптинга у рушії використовується власна мова - UnrealScript. На сайті розробників представлені тонни навчальних матеріалів, як текстових, так і відео, як по редактору, так і по скриптингу. UE3 отримав безліч нагород на індустріальних заходах, а також в кінематографі і не раз ставав кращим ігровим / графічним рушіем року.

unity3d


Офіційний сайт розробника: unity3d.com

Найпопулярніший рушій для створення 2D- і 3D-ігор. Безперечно, він став лідером індустрії, і, як тільки з'являється нова ігрова / графічна технологія, розробники негайно реалізують її в Unity. Крім розробки сінглплеерних ігор для PC, за допомогою підключаються експортерів можна перенести гри під інші ОС, консолі і мобільні технології (за експортер доведеться доплатити 1500 доларів за кожну платформу: iOS, Android, BlackBerry). Плюс до цього утворилася ціла індустрія, що працює над створенням доповнень і розширень рушія, серед них є як спеціалізовані серверні рішення для Unity (e. g. Photon - повноцінний ігровий сервер), так і засоби для розробки користувальницького інтерфейсу (NGUI), конструктори, призначені для створення гри певних жанрів (e. g. Playmaker).
В самого редактора Unity є порти під OS X і Windows, при цьому спочатку він був призначений для OS X. В Unity включена підтримка DirectX 11, що відкриває твоїм додатків дорогу в світи Windows 8 і Windows Phone 8. У версії Unity під номером 4.2, з'явилася підтримка останньої на даний момент OpenGL ES 3.0, поки цими засобами мають тільки топові Android-смартфони. Рушій Unity особливо цінний за низький поріг входження для початківців користувачів, завдяки цього, а також того, що інді-версія безкоштовна, навколо движка зорганізувалось величезне співтовариство. Низький поріг входження є результатом грамотного дизайну програми: багато речей можна виконати за допомогою різних редакторів, не написавши при цьому жодного рядка коду (якщо що, код пишеться на JavaScript, C #, Boo). Вихідний код на C / C ++ закритий, але це у зв'язку з розширенням компонентної структури рушія і не створює ніяких перешкод.
